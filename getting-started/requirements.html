System Requirements

Detailed system requirements and compatibility information.

Minimum Requirements

  • Operating System: Windows 10, macOS 10.15, or Linux (Ubuntu 18.04+)
  • RAM: 4GB minimum, 8GB recommended
  • Storage: 1GB available space
  • Network: Internet connection required for initial setup

Supported Platforms

This project is tested and supported on the following platforms:

  • Windows 10/11
  • macOS 10.15+
  • Ubuntu 18.04+
  • CentOS 7+
