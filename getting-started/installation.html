Installation

This comprehensive guide covers installation instructions for all RoboGPT software versions and repositories.

Overview: RoboGPT is available in two main versions:
  • RoboGPT v3 - Dead
  • RoboGPT v4 - Alive (Latest version with cutting-edge features)

‚ö° RoboGPT v4 (Development)

Latest version of RoboGPT with all the new tech and some wow type features also known as cutting-edge features.

Installation Steps

# 1. Clone the main repository git clone https://github.com/orangewood-co/robogpt.git cd robogpt # 2. Clone additional repositories in the same workspace git clone https://github.com/orangewood-co/robogpt_tools.git git clone https://github.com/orangewood-co/robogpt_perception.git git clone https://bitbucket.org/owl-dev/orangewood_simstack_ros2.git git clone https://github.com/orangewood-co/robogpt-agentic-workflow.git git clone https://github.com/orangewood-co/prompt_validator.git git clone https://github.com/orangewood-co/robogpt_external_hardware.git # 3. Clone third-party packages git clone https://github.com/orbbec/OrbbecSDK_ROS2.git git clone https://github.com/AndrejOrsula/pymoveit2.git
